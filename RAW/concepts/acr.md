# Agent Control Room (ACR)

ACR = Agent Control Room

The Agent Control Room is the internal mission control interface for AiAgentNerd.

It is used to:
- visualize all agents
- monitor active tasks
- manage users
- control execution
- view system activity

ACR consists of:
- user panel
- agent office view
- task panel
- activity feed
- system monitoring panel
