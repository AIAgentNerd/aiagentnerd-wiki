# Deck

## Summary
Deck is the primary control interface for the AiAgentNerd multi-agent system.

It provides a unified operational view to:
- visualize agents
- monitor tasks
- manage users
- observe real-time system activity

---

## Concept

Deck represents the control surface of the system using a ship metaphor:

- AiAgentNerd = the ship  
- Deck = the control interface  
- agents = the crew  
- tasks = missions  
- activity feed = the ship log  
- system monitoring = ship status  

---

## Core Functions

Deck enables:

- **Agent visualization**  
  Real-time view of all agents and their current status

- **Task monitoring**  
  Tracking of active and queued tasks

- **User management**  
  Control of users and permissions

- **System activity observation**  
  Chronological event stream of system actions

---

## Role in System

Deck acts as:

- the primary control interface for AiAgentNerd  
- the visualization layer of Hermes task execution  
- the operator’s entry point into the system
