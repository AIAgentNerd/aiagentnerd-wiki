# Deck

Deck is the control interface of AiAgentNerd (formerly referred to as ACR).

It is the central place to:
- visualize agents
- monitor tasks
- manage users
- observe system activity

Metaphor:
- AiAgentNerd = ship
- Deck = control surface
- agents = crew
- tasks = missions
- activity = ship log
