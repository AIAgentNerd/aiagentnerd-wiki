# AionUi Chat Exchange

## User

what is deck in aiagentnerd ?

## Assistant

Deck is AiAgentNerd’s internal web dashboard, accessible at `deck.aiagentnerd.com`. It’s a single-page interface for monitoring and managing the multi-agent system in real time. The layout consists of:

- **User Panel** – lists users and their active task counts, with filtering.
- **Agent Office View** – shows the full agent crew, each agent’s current task, status, and model.
- **Task Panel** – displays active and queued tasks with ID, type, user, and status; supports admin stop/retry.
- **Activity Feed (Ship Log)** – a chronological log of system events, agent actions, and errors.
- **System Monitoring Panel** – live CPU, memory, health, and alerts.

Updates happen via polling or WebSocket. Deck serves as the primary control interface, the visualization layer for Hermes task execution, and the operator’s entry point into the system.
