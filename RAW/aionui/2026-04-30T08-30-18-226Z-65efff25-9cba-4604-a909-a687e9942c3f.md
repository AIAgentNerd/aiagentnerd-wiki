# AionUi Chat Exchange

## User

What is Deck in AiAgentNerd?

## Assistant

Deck is the operational dashboard of AiAgentNerd—the internal web interface for monitoring and controlling the multi-agent system in real time. It lives at `deck.aiagentnerd.com` and gives you a live view of everything happening under the hood.

**What it shows:**
- **User Panel**: which users are active and how many tasks they have running
- **Agent Office View**: every agent (the crew), their current task, status, and the model they're using
- **Task Panel**: all active and queued tasks, with controls to stop or retry
- **Activity Feed (Ship Log)**: a chronological log of system events, agent actions, and errors
- **System Monitoring**: CPU, memory, health, and alerts

**How it works:**
- It pulls in structured data (`users`, `agents`, `tasks`, `activity`, `system_status`)
- You can click on any agent to see their task, filter by user, or drill into a task's details
- Updates come in live via polling or WebSocket

**Its role:**
Deck is the primary control surface for AiAgentNerd. It's where the operator (or anyone monitoring the system) sees what Hermes is doing, manages the agent fleet, and keeps an eye on system health. Think of it as the bridge between the human operator and the autonomous agent system.
