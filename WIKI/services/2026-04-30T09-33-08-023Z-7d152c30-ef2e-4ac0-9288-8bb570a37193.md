# Aionui 2026 04 30T09 33 08 023Z 7d152c30 Ef2e 4ac0 9288 8bb570a37193

## Summary
Deck is the operational dashboard and primary control interface for the AiAgentNerd multi-agent system. It is a single-page web application hosted at `deck.aiagentnerd.com` that provides real-time monitoring and management of agents, tasks, users, and system health.

## Key Concepts
- **Deck**: Single-page operational dashboard and visualization layer for Hermes task execution
- **Five-panel layout**: User Panel, Agent Office View, Task Panel, Activity Feed (Ship Log), and System Monitoring
- **Real-time updates**: Live state streaming via polling or WebSocket
- **Operator entry point**: Primary interface for managing the multi-agent crew

## Practical Use
- Navigate to `deck.aiagentnerd.com` to monitor the multi-agent system
- Use the **Agent Office View** to see each agent's current task, status, and active model; this is the primary interaction area
- Use the **User Panel** to list users, view active task counts per user, and filter tasks by user
- Use the **Task Panel** to review active and queued tasks (task ID, type, user, status) and execute admin controls to stop or retry jobs
- Use the **Activity Feed (Ship Log)** to review chronological system events, agent actions, and error messages
- Use the **System Monitoring** panel to check CPU, memory, health metrics, and runtime alerts

## Implementation Notes
- **Data model**: Deck consumes structured data arrays including `users`, `agents`, `tasks`, `activity`, and `system_status`
- **Interactions**:
  - Click an agent → reveals its current task
  - Click a user → filters tasks by that user
  - Click a task → opens task details
  - Admin controls → stop or retry tasks directly from the interface
- **Real-time behavior**: Task states, agent state changes, and activity stream updates are delivered live through polling or WebSocket
- **System role**: Serves as the primary control interface for AiAgentNerd, the visualization layer for Hermes task execution, and the operator's main entry point into the system
