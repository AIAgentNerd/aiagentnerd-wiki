# Karpathy Style LLM Wiki

Here's a clear, step-by-step guide to set up your Obsidian wiki with a clean Topic Brain input system using Web Clipper:

---

### **Step 1: Create Your Folder**
1. In Obsidian, go to **Vaults**.
2. Click **New Folder**.
3. Name it: `raw/web` (this is your input folder)
4. Click **Create**.

---

### **Step 2: Clip Your Note**
1. Open **Web Clipper** (search in apps).
2. Click the **Web Clipper icon** in the top toolbar.
3. In the Web Clipper settings:
   - Click **Add to Obsidian**.
   - Select **raw/web** (ensure it’s listed).
4. Click **Save**.

---

### **Step 3: Verify the Clip**
1. Open your `raw/web` folder.
2. Find your clipped note (e.g., "X Post Idea").
3. Check it appears in Obsidian’s sidebar or page.

---

### **Step 4: Finalize Your Setup**
1. **Clean up**: Delete any old clippings.
2. **Test**: Add a new note to `raw/web` to confirm it works.
3. **Label clearly**: Use a consistent folder name (e.g., `Projects`, `Personal`).

---

### **Why This Works**
- **`raw/web`**: Isolates your input for easy access.
- **Web Clipper**: Automates clipping (no manual copy-pasting).
- **Simple folder structure**: No clutter, easy navigation.

---

### **Troubleshooting Tips**
- **Error?** Ensure `raw/web` exists. Check Web Clipper settings.
- **Note missing?** Re-clip or check folder permissions.
- **Want more?** Add a `.md` file to `raw/web` for quick notes.
