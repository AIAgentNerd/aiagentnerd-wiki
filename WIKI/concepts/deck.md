# Concepts Deck

## Summary
Deck is the primary control interface for the AiAgentNerd multi-agent system. It provides a unified operational view to visualize agents, monitor tasks, manage users, and observe real-time system activity, using a ship metaphor for clarity.

## Key Concepts
- Ship metaphor: AiAgentNerd is the ship, Deck is the control surface, agents are the crew, tasks are missions, activity feed is the ship log, system monitoring is ship status.
- Agent visualization: real-time view of all agents and their status.
- Task monitoring: tracking active and queued tasks.
- User management: control of users and permissions.
- System activity observation: chronological event stream of system actions.

## Practical Use
- Use Deck as the operator’s entry point to monitor and control the multi-agent system.
- View current agent assignments and status in the agent visualization panel.
- Manage task queues, including stop/retry tasks from the task monitoring view.
- Check system health and event history through the activity feed and monitoring panels.
