## Summary
The Agent Control Room (ACR) is AiAgentNerd’s internal mission-control dashboard — a single pane of glass for monitoring and managing the entire agent ecosystem. It displays real-time agent statuses, task queues, user permissions, an activity log, and system health metrics, giving operators a unified operational view and the ability to intervene manually when needed.

## Key Concepts
- Centralised dashboard for agent operations
- Live agent visualisation (statuses: idle, working, etc.)
- Real-time task monitoring (active and queued)
- User and permission management
- Chronological activity feed of agent actions and system events
- System health metrics and alerting

## Practical Use
- Use ACR to observe overall system state without directly accessing logs or databases
- Perform manual interventions: start/stop agents, adjust workflows, inspect stalled tasks
- Surface system anomalies through health metrics and alerts
